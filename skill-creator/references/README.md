# References Directory

This directory contains detailed documentation and reference materials for the skill-creator.

## Purpose

The references/ folder is part of the progressive disclosure pattern. Files here are loaded on demand when detailed information is needed, keeping the main SKILL.md concise (under 2,000 words).

## Contents

- Extended guides (2,000-5,000 words)
- Technical deep-dives
- Best practice documents
- Style guides
- Validation checklists

## Usage

Reference files are automatically discovered by the runtime when requested via:
- Skill introspection
- Explicit user requests
- Template loading

*This directory is intentionally minimal. Add relevant reference files as needed.*
