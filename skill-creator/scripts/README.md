# Scripts Directory

This directory contains executable utilities for skill-creator maintenance.

## Purpose

The scripts/ folder provides automation tools for managing, validating, and updating the skill.

## Contents

- Validation scripts
- Update utilities
- Generation tools
- Maintenance scripts

## Usage

Scripts are executable utilities that can be run directly or invoked by the skill when needed.
